# Data (Course-provided)

The raw dataset used in this project was provided through a course and is not included in this repository.

To reproduce results:
1. Obtain the course-provided files (same filenames as used in the Rmd).
2. Place them under the `data/` folder.
3. Re-run `Final_Project.rmd`.

Note: The large file `us-counties-all.csv` is excluded from version control due to GitHub file size limits.
